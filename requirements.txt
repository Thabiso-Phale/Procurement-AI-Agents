# Procurement AI Agent Suite

Four AI-powered procurement agents in one web app, powered by Claude AI.

## Agents
- 📄 RFQ / RFP Drafter
- 📊 Bid Evaluator
- ⚖️ Contract Reviewer
- 💰 Spend Analyser

---

## Deploy to Railway (5 minutes, free to start)

### Step 1 — Get your Anthropic API key
1. Go to **console.anthropic.com**
2. Sign in (or create a free account)
3. Click **API Keys** in the left sidebar
4. Click **Create Key** → copy the key (starts with `sk-ant-...`)

### Step 2 — Deploy to Railway
1. Go to **railway.app** and sign in with GitHub
2. Click **New Project** → **Deploy from GitHub repo**
3. Select this repository
4. Click **Add Variables** and add:
   - Key: `ANTHROPIC_API_KEY`
   - Value: paste your key from Step 1
5. Click **Deploy** — Railway builds and launches automatically
6. Click the generated URL — your app is live!

## Run locally (optional)

```bash
pip install -r requirements.txt
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
python app.py
# Open http://localhost:5000
```

Once live, add subscription paywalls using Stripe + Whop:

2. Set up a Whop store and paste your app URL as the member resource
3. Members get access via Whop after paying through Stripe
